# 🚀 ¡Bienvenido a @scib/ui-kit!

La librería de componentes UI para tus proyectos Angular, React y más. Todo lo que necesitas para empezar, en un solo lugar.

---

## 1️⃣ Instalación rápida

```bash
npm install -D @scib/ui-kit@^3.0.0
```

---

## 2️⃣ Ejemplo básico

<div style={{display:'flex', flexDirection:'column', gap:'24px', width:'100%'}}>
  <div style={{border:'1px solid #e0e0e0', borderRadius:'8px', padding:'20px', background:'#fafbfc', width:'100%'}}>
    <strong style={{color:'#1EA7FD', fontSize:'16px'}}>React</strong>
    <pre style={{background:'#f8f8f8', borderRadius:'6px', padding:'12px', margin:'12px 0', fontSize:'14px', overflow:'auto'}}>
{`import { ScibAtomsButton } from '@scib/ui-kit/react';

export function Demo() {
  return <ScibAtomsButton color="primary">¡Hola UI Kit!</ScibAtomsButton>;
}`}
    </pre>
    <div style={{marginTop:'8px', fontSize:'13px', color:'#b71c1c', background:'#fff3e0', border:'1px solid #ffe0b2', borderRadius:'4px', padding:'8px 12px'}}>
      <strong>Nota:</strong> Para que los componentes funcionen correctamente en React, es necesario añadir <code>"importHelpers": true</code> en el <code>tsconfig.json</code> del proyecto donde uses la librería.
    </div>
  </div>
  <div style={{border:'1px solid #e0e0e0', borderRadius:'8px', padding:'20px', background:'#fafbfc', width:'100%'}}>
    <strong style={{color:'#1EA7FD', fontSize:'16px'}}>Angular</strong>
    <pre style={{background:'#f8f8f8', borderRadius:'6px', padding:'12px', margin:'12px 0', fontSize:'14px', overflow:'auto'}}>
{`// app.module.ts
import { ScibAtomsButtonModule } from '@scib/ui-kit/angular';

@NgModule({
  imports: [
    ScibAtomsButtonModule,
    // ...otros módulos
  ],
  // ...
})
export class AppModule {}`}
    </pre>
    <pre style={{background:'#f8f8f8', borderRadius:'6px', padding:'12px', margin:'12px 0', fontSize:'14px', overflow:'auto'}}>
{`<!-- En tu template HTML -->
<scib-button color="primary">¡Hola UI Kit!</scib-button>`}
    </pre>
  </div>
</div>
