# 🚀 Migration Guide v3+

¡Bienvenido a la nueva versión de `@scib/ui-kit`! A partir de la versión **3.0.0** todo es mucho más sencillo: ya no necesitas configurar `includes` ni `paths` en tu `tsconfig`. Solo necesitas instalar el paquete y empezar a importar.

---

## 🧹 1. Desinstala versiones antiguas

Si tienes versiones previas de los paquetes wrappers, elimínalos:

```bash
npm uninstall @scib/ui-kit
```

---

## 📦 2. Instala la nueva versión

Instala la versión mínima recomendada (3.0.0 o superior):

```bash
npm install -D @scib/ui-kit@^3.0.0
```

---

## 🧽 3. Limpia tu configuración de TypeScript

Elimina cualquier referencia a los antiguos includes y paths relacionados con `@scib/angular-ui-kit`, `@scib/react-ui-kit` o rutas personalizadas de `@scib/ui-kit` en tus archivos `tsconfig.json` y `tsconfig.app.json`.

### Ejemplo de limpieza en `tsconfig.app.json`:

- Elimina líneas como:

```jsonc
"include": [
  "node_modules/@scib/angular-ui-kit/**/*",
  "node_modules/@scib/react-ui-kit/**/*",
  "node_modules/@scib/ui-kit/dist/angular/**/*",
  "node_modules/@scib/ui-kit/dist/react/**/*",
  "node_modules/@scib/ui-kit/dist/types/components/**/*"
]
```

- Deja solo lo necesario, por ejemplo:

```jsonc
"include": [
  "src/**/*.d.ts",
  "src/main.ts",
  "src/polyfills.ts"
]
```

### Ejemplo de limpieza en `tsconfig.json`:

- Elimina cualquier entrada en `paths` como:

```jsonc
"paths": {
  "@scib/angular-ui-kit": ["node_modules/@scib/ui-kit/dist/angular/index.ts"],
  "@scib/react-ui-kit": ["node_modules/@scib/ui-kit/dist/react/index.ts"]
}
```

- Si no tienes otros paths personalizados, puedes dejar el objeto vacío o eliminar la clave `paths` si no es necesaria.

---

## ✨ 4. Importa directamente desde `@scib/ui-kit`

Ahora puedes importar cualquier componente, hook o utilidad directamente:

```typescript
import { ScibAtomsButtonModule } from '@scib/ui-kit/angular';
```

No necesitas modificar tu `tsconfig.json` ni añadir rutas personalizadas.

---

## 📝 5. ¡Listo para usar!

Ya puedes usar todos los componentes y utilidades del kit sin configuraciones adicionales. Por ejemplo:

```tsx
<scib-atoms-button lavel="button">Hello UI Kit!</scib-atoms-button>
```

---

## 🎉 Ventajas de la nueva versión

- Sin configuración extra de TypeScript
- Imports unificados y simples
- Menos dependencias y menos errores
- Actualizaciones más fáciles

### Example cleanup in `tsconfig.app.json`:

```jsonc
"include": [
```
