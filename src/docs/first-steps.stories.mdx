## Getting Started

La primera vez que se instale el repo, es necesario instalar las dependencias

```bash
npm install
```

Para lanzar el entorno de desarrollo local:

```bash
npm start
```

Se levantará un servidor que inifica todos los contenidos:

http://localhost:4300/ -> Storybook, para documentación y desarrollo en el canvas de aislamiento y pruebas visuales

Se generar un doc específico de playgrounds para acceder a la visualización de los mismos en una pestaña nueva
