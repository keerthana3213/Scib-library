#### No me actualiza el componente cuando hago cambios en su atributo en el DOM

Puede ser porque el nombre del atributo sea el incorrecto. Las `@Prop` se nomenclan en camelCase, pero los atributos de HTML siempre deben ir en kebab-case:

```typescript
export class MyComponent {
  @Prop() myCustomProp: string; 
  ...
}
```

```html
<my-component my-custom-prop="test"></my-component>
```

#### Cuando paso un objeto o un array como JSON string a una @Prop(), no se parsea

Stencil no parsea automáticamente strings JSON a objetos/arrays, hay que hacerlo manualmente y mandar el objeto parseado a un @State() para tenerlo disponible:
```typescript
  class MyComponent {

    /** Listado de notificaciones a ser mostradas */
    @Prop() notifications: string | ICDKNotificationAreaItem[];

    /** Escucha los cambios en la prop */
    @Watch('notifications') notificationsChange(newVal: string | ICDKNotificationAreaItem[]) { 
      this._notifications = _parseProp(newVal);
    }

    @State() _notifications

  }
  
```
